Configuración de una Red Básica

Laboratorio práctico paso a paso en Cisco Packet Tracer.

Teoría del Laboratorio

En este módulo aprenderás a interconectar dispositivos finales a través de un switch y configurar una puerta de enlace (Gateway) en un router. Utilizaremos el segmento de red 192.168.10.0 con una máscara de subred de 24 bits (255.255.255.0).

Elementos de la Topología:

  • Router 2911: Encargado de enrutar el tráfico fuera de nuestra red local.
  • Switch 2960: Conecta físicamente los equipos de la LAN.
  • PC de Escritorio y Laptop: Equipos terminales que consumirán y enviarán datos en la red.

Tabla de Direccionamiento Estático

Configura los parámetros de red en las propiedades de cada dispositivo en Packet Tracer según los siguientes datos:

Dispositivo Interfaz Dirección IP Máscara de Subred Gateway (Puerta de enlace)
Router 2911 GigabitEthernet 0/0 192.168.10.1 255.255.255.0 N/A
Switch 2960 Interfaces FastEth Sin IP (Capa 2) N/A N/A
PC Escritorio FastEthernet 0 192.168.10.2 255.255.255.0 192.168.10.1
Laptop FastEthernet 0 192.168.10.3 255.255.255.0 192.168.10.1

Comandos de Consola (CLI) para el Router 2911

Haz clic en el Router, ve a la pestaña CLI y digita los siguientes comandos para levantar la interfaz:

Router> enable
Router# configure terminal
Router(config)# interface gigabitEthernet 0/0
Router(config-if)# ip address 192.168.10.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# end
Router# copy running-config startup-config
